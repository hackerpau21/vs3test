# Test Theme 7 - Timeless Blog Style

A timeless, professional WordPress theme with elegant design inspired by Maria Orozova. Features clean layouts, custom Gutenberg blocks, and mobile-first responsive design.

## Features

### Design & Layout
- **Timeless Design**: Clean, minimalist aesthetic that won't feel dated
- **Professional**: Sophisticated styling suitable for personal brands and thought leaders
- **Mobile-First**: Responsive design that works seamlessly across all devices
- **Hero Section**: Full-viewport background image with overlay text
- **Card Grid Layout**: Beautiful 3-column grid for showcasing content

### Customization Options

## Installation

1. **Download** the theme files
2. **Upload** to your WordPress site via:
   - Admin Dashboard → Appearance → Themes → Add New → Upload Theme
   - Or upload the folder directly to `/wp-content/themes/`
3. **Activate** the theme from Appearance → Themes
4. **Customize** via Appearance → Customize

## Theme Setup

### Initial Configuration

1. **Set Up Menus**:
   - Go to Appearance → Menus
   - Create a new menu and assign it to "Primary Menu"
   - Optionally create a footer menu

2. **Configure Hero Section**:
   - Go to Appearance → Customize → Hero Section
   - Upload a background image (recommended: 1920x1080px)
   - Set your hero title and description

3. **Social Media Links**:
   - Go to Appearance → Customize → Social Media Links
   - Add your social media URLs

4. **Widgets**:
   - Go to Appearance → Widgets
   - Configure footer widget areas as needed

### Recommended Pages

Create these essential pages for best results:
- **Home** (set as front page)
- **Blog** (set as posts page)
- **About**
- **Contact**
- **Privacy Policy**

- **Headers**: Playfair Display (serif) or Montserrat (sans-serif)
